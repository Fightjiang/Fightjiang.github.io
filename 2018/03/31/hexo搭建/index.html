Linux下搭建 hexo

我想每个人都希望的是拥有一个属于自己的博客的吧，没错在这里我将为你提供的是 github + hexo 来搭建属于自己的一个博客！

１．安装hexo 所需要的依赖

Sudo apt-get install git nodejs npm

2. 安装hexo

sudo npm install -g hexo

3.在你的用户目录下创建 blog

mkdir ~/blog

4.进入blog目录

cd ~/blog

5.初始化hexo目录

hexo init

6.生成并运行hexo

hexo s
然后会生成这样的提示：

它告诉我们可以到 http://localhost:4000/ 这个链接下去访问，好，我们去看看。

很好，搭建成功了的啦！
赶快去看看的吧！
